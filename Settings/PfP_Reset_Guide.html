Part 2 — Step by Step Reset

Step 1 — Clear the LEDGER tab

  1. Open your Google Sheet and click the LEDGER tab.
  2. Click row 2 to select it, then hold Shift and click the last row with data.
  3. Right-click → Delete rows.
  4. Row 1 (the header) stays. The sheet should now be empty below the header.

Step 2 — Reset the card record in CARDS tab

  1. Click the CARDS tab and find your test card row (EVO-ART-1-108).
  2. Clear the following columns for that row:
    • Claimed — change back to NO
    • Claim Date — delete the value
    • Owner Token — delete the value
    • Current XP — set to 0
    • Current Level — set to 1
    • Evolution Stage — set to Dormant
    • Streak (days) — set to 0
    • Last Interaction — delete the value
    • Total Interactions — set to 0
    • Page Views — set to 0

Step 3 — Clear OWNERS, OWNERSHIP HISTORY, MILESTONES, Gaming tabs

  1. For each of these tabs, select all data rows below the header and delete them the same way as Step 1.
  2. Leave the header row intact in each tab.

Step 4 — Clear browser localStorage on your device

This removes the saved owner token so the dashboard treats the card as unclaimed on your browser.

  1. Open the dashboard in Chrome.
  2. Press F12 (Windows) or Cmd+Option+I (Mac) to open DevTools.
  3. Click the Application tab.
  4. In the left panel expand Local Storage and click your site's URL.
  5. Find any key starting with pfp_owner_ and delete it.
  6. Also delete any key starting with pfp_viewed_.

Step 5 — Verify the reset

  1. Open the dashboard with your card's URL.
  2. It should show the claim screen asking for a claim code — not the owner dashboard.
  3. Open Your Life — it should show "No Life Events Recorded Yet."
  4. The system is now clean and ready for a real owner to claim the card.
